Obtain links for upload and get file. IMPORTANT: For each upload must generate a new link!

SHORT API INSTRUCTION

Please call this API as HTTP POST method with replacing of the following parameters:

  • $AUTH_TOKEN - your client token
  • $PASSWORD - your password
  • $SYNC MODE - following strings are allowed
    • full - to REPLACE audience ID's
    • iter-add - to ADD new audience ID's
    • iter-delete - to REMOVE unneeded audience ID's

Server will return JSON, that will contain a links for upload and get file:

Example:
{
"upload_location": "https://link.for.upload?some=params",
"get_upload":"https://link.for.get?some=params",
"expires":"XXXX-XX-XXT00:00:00"
}

  • Put URL(upload_location) - to upload/add/remove your ID's based on $SYNC MODE parameter value
  • Get URL(get_upload) - to upload currently available ID's loaded by you before
  • Exprires - expiration time, when Post and Get URL's will become invalid.

Files can be loaded with CURL logic, sample:
curl --upload-file '/file_path/file_name.csv.gz' upload_location

Please Note:

  • Allowed file formats - CSV/TXT, then need to compress to .gz
  • Just added Audience will start to work in 3 hours
